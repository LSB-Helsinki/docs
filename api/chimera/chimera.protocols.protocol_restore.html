chimera.protocols.protocol_restore module¶

class chimera.protocols.protocol_restore.ChimeraProtRestore(**kwargs)[source]¶

Bases: chimera.protocols.protocol_base.ChimeraProtBase

This protocol opens Chimera and restores a session that has been stored each time a 3Dmap or an atomic structure by using scipionwrite or scipionss commad.

Execute command scipionwrite [model #n] [refmodel #p] [saverefmodel 0|1] from command line in order to transfer fitted pdb to scipion. Default values are model=#0, refmodel =#1 and saverefmodel 0 (false). model refers to the pdb file. refmodel to a 3Dmap
